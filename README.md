# Карта памятников культуры Алтайского края

Интерактивная веб-карта с объектами культурного наследия Алтайского края на основе официального реестра.

### Демо
Посмотреть карту:  
**[Ссылка на GitHub Pages](https://egrvtkn.github.io/altay-heritage-map/)**  
(или просто открой `index.html` локально через Live Server)

### Что внутри проекта

- **Данные**: ~700 объектов культурного наследия Алтайского края (очищенные и геокодированные)
- **Технологии**:
  - Leaflet.js — интерактивная карта
  - OpenStreetMap / CartoDB — подложка
  - GeoJSON — формат данных
  - Python (pandas, geopy) — очистка и геокодирование
